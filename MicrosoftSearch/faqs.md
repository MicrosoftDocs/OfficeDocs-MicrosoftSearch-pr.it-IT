# <a name="faqs"></a>Domande frequenti

Queste sono le domande frequenti relative alle organizzazioni e gli utenti con sull'amministrazione e utilizzo di Microsoft Search.
  
## <a name="whats-the-value-proposition-for-microsoft-search"></a>Che cos'è il valore della proposta di Microsoft Search?

Microsoft Search è un modo sicuro per il lavoro e contenuto web di ricerca. Questo tipo di integrazione tra web ed enterprise è disponibile solo con Microsoft. Per informazioni relative alla conformità, vedere [Security for Microsoft Search](security.md).
  
## <a name="what-microsoft-search-features-are-available-now"></a>Ora sono disponibili le funzionalità di Microsoft Search?

Per un elenco completo, vedere [Funzionalità di ricerca di Microsoft](features.md).
  
## <a name="does-microsoft-search-support-advanced-query-understanding"></a>Informazioni sulle query di supporto avanzato Microsoft Search non?

Sì, Microsoft Search analizza lo scopo di query da più frasi. Questa funzionalità utilizza per scoprire comuni agli utenti di frasi superfluo aggiungere per le query che non influiscono le finalità di ricerca. Ad esempio, quando un utente cerca 'ulteriori informazioni su come modificare la password,' è estrarre le parole meno importante da query e trigger basati su quelli pertinenti like 'cambiare la password'.
  
Questa funzionalità non sostituisce il set di parole chiave nel portale di amministrazione.
  
## <a name="does-microsoft-search-search-for-files-on-premises-as-well-as-the-cloud"></a>Microsoft Search cerca i file locali e su come cloud?

Microsoft Search supporta la ricerca di documenti che risiedono su SharePoint Online, OneDrive for Business, nonché su ibrido locale di SharePoint e SharePoint Online per i tipi di file più comuni di Office. Se è possibile cercare il contenuto locale in SharePoint Online, deve essere in grado di trovare contenuto locale tramite Microsoft Search. 
  
## <a name="does-microsoft-search-replace-other-enterprise-search-experiences"></a>Microsoft Search sostituisce altre esperienze di ricerca dell'organizzazione?

Microsoft Search è un'offerta complementare che combina i risultati da più origini. Per ulteriori informazioni sui tipi di file supportati e le origini, vedere le domande frequenti.
  
## <a name="what-file-types-and-sources-does-microsoft-search-support"></a>Quali tipi di file e le origini supportati da Microsoft Search?

Si supportano le origini di contenuto seguenti:
  
- SharePoint Online
    
- Ibrido di SharePoint (locale + SharePoint Online)
    
- OneDrive for Business
    
I seguenti tipi di file visualizzati nella ricerca di file e vengono visualizzati nella scheda file per utenti e gruppi:
  
- Word
    
- Excel
    
- PowerPoint
    
- OneNote
    
- PDF
