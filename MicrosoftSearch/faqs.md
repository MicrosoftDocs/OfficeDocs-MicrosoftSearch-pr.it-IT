# <a name="faqs"></a>Domande frequenti

Ecco le domande frequenti poste da organizzazioni e utenti sulla gestione e sull'uso di Microsoft Search.
  
## <a name="whats-the-value-proposition-for-microsoft-search"></a>Qual è la proposta di valore di Microsoft Search?

Microsoft Search è un modo sicuro per effettuare la ricerca di contenuti Web e di lavoro. Questo tipo di integrazione tra Web e organizzazione è disponibile solo con Microsoft. Per informazioni sulla conformità, vedere [Sicurezza per Microsoft Search](security.md).
  
## <a name="what-microsoft-search-features-are-available-now"></a>Quali funzionalità di Microsoft Search sono ora disponibili?

Per un elenco completo, vedere [Funzionalità di Microsoft Search](features.md).
  
## <a name="does-microsoft-search-support-advanced-query-understanding"></a>Microsoft Search supporta il riconoscimento avanzato delle query?

Sì. Microsoft Search analizza l'intento della query da frasi più lunghe. Questa funzionalità usa l'intelligenza artificiale per apprendere le frasi superflue comuni che gli utenti aggiungono alle query e che non influiscono sull'intento di ricerca. Ad esempio, quando un utente cerca "altre informazioni su come cambiare la password", vengono estratte le parole meno importanti dalla query e la ricerca viene attivata in base a quelle più pertinenti, come "cambiare password".
  
Questa funzionalità non sostituirà le parole chiave impostate nel portale di amministrazione.
  
## <a name="does-microsoft-search-search-for-files-on-premises-as-well-as-the-cloud"></a>Microsoft Search cerca i file in locale oltre che sul cloud?

Microsoft Search supporta la ricerca di documenti presenti in SharePoint Online, OneDrive for Business, oltre che in ambiente ibrido locale SharePoint e SharePoint Online per i vari tipi di file di Office. Se è possibile eseguire ricerche di contenuto locale in SharePoint Online, sarà anche possibile cercare contenuto locale con Microsoft Search. 
  
## <a name="does-microsoft-search-replace-other-enterprise-search-experiences"></a>Microsoft Search sostituisce altre esperienze di ricerca di contenuti nell'organizzazione?

Microsoft Search è un'offerta complementare che raggruppa i risultati provenienti da più origini. Per altre informazioni sulle origini e sui tipi di file supportati, vedere le domande frequenti.
  
## <a name="what-file-types-and-sources-does-microsoft-search-support"></a>Quali tipi e origini di file supporta Microsoft Search?

Microsoft supporta le origini di contenuto seguenti:
  
- SharePoint Online
    
- SharePoint ibrido (locale + SPO)
    
- OneDrive for Business
    
I tipi di file seguenti vengono restituiti dalla ricerca file e visualizzati nella scheda File per Utenti e gruppi:
  
- Word
    
- Excel
    
- PowerPoint
    
- OneNote
    
- PDF

Le ricerche di Microsoft Search non permettono di guadagnare punti Rewards e non sono mai associate all'account Microsoft di un utente.

## <a name="how-does-microsoft-search-use-azure-active-directory"></a>In che modo Microsoft Search usa Azure Active Directory?

Microsoft Search usa Azure Active Directory per autenticare e autorizzare l'accesso ai dati dell'organizzazione. Microsoft Search è conforme alle pratiche standard di gestione delle identità in tutti i prodotti Microsoft. Questo significa che gli utenti possono accedere automaticamente con il servizio Single Sign-On.

## <a name="does-microsoft-search-provide-search-results-across-tenants"></a>Microsoft Search restituisce risultati della ricerca da tutti i tenant?

## <a name="why-am-i-signed-into-bing-automatically"></a>Perché si è connessi automaticamente a Bing?

Microsoft Search usa il Single Sign-On di AAD per effettuare automaticamente l'accesso con l'account aziendale o dell'istituto di istruzione quando si è connessi a un'applicazione di Office 365 oppure a Windows 10. È possibile disconnettersi in qualsiasi momento.
